# Looped Automation Homebrew Tap

Homebrew casks for [Looped Automation](https://looped.sh) macOS apps.

## Looped Whisper

Free, open-source, local voice transcription for macOS.

```bash
brew install --cask loopedautomation/tap/looped-whisper
```

> Requires Apple Silicon, macOS 14 (Sonoma) or later.

The cask is updated automatically by the
[whisper](https://github.com/loopedautomation/whisper) release workflow.
